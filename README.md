Student Management System
Overview
The Student Management System is a Python-based application that helps manage student records efficiently. It allows users to add, update, delete, and view student information in an organized manner.
Features
Add new student records
View student details
Update existing records
Delete student records
Search for students
User-friendly menu-driven interface
Technologies Used
Python 3
Installation
Clone the repository:
Bash
git clone <repository-url>
Navigate to the project folder:
Bash
cd student-management-system
Run the program:
Bash
python main.py
Usage
Launch the application.
Select options from the menu to manage student records.
Enter the required details when prompted.
